kt.jsonapi
==========

.. toctree::
    :maxdepth: 5

``kt.jsonapi`` supports generation of `JSON:API`_ responses using
adaptation from application objects.

The current implementation works with the Flask_ web framework.


.. _Flask:
   https://flask.palletsprojects.com/

.. _JSON\:API:
   https://jsonapi.org/
